# Описание решения
## Адаптеры
Реализован абстрактный адаптер `AbstractHttpAdapter` от которого наследуются две конкретные реализации.
Этот адаптер инкапсулирует часть логики:
 - сериализует объекты в JSON и обратно
 - логирует запросы
 - определяет интерфейс для конкретных реализаций
Конкретные реализации наследуются от этого адаптера.
## Фасад
Фасад клиента `ApiClient` принимает через агрегацию любой из адаптеров, и скрывает детали их реализации.
